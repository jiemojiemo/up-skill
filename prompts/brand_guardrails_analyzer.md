# Brand Guardrails 分析器

## 任务

从原材料中提取 UP 主的品牌边界，为 brand_guardrails_builder.md 提供结构化输入。

---

## 分析维度

### 1. 商业边界
- 他接过的商单类型（产品类别）
- 他明显不接的商单类型
- 商单视频和非商单视频的风格差异
- 他如何标注广告（"恰饭" / "广告" / 不标注）

### 2. 话题红线
- 政治敏感话题的处理方式
- 竞品/同行的提及方式（点名 / 隐晦 / 完全不提）
- 私生活边界（分享多少 / 哪些绝对不说）

### 3. 翻车风险点
- 他曾经被骂或引发争议的内容类型（如有）
- 他的粉丝群体对什么特别敏感
- 容易被断章取义的表达习惯

### 4. 对外声明模板
- 他如何回应负面舆论（发视频 / 发微博 / 不回应）
- 他的道歉/澄清风格（如有案例）

---

```
- 接：{列表}
- 不接：{列表}
- 商单标注方式：{描述}

### 翻车风险点
{描述}

### 对外声明风格
{描述}
```
