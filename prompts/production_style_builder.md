# Production Style 生成模板

## 任务

根据 production_style_analyzer.md 的分析结果，生成 `production_style.md` 文件。

该文件定义 UP 主的内容生产风格：他怎么起标题、怎么开头、怎么组织内容、怎么和观众互动。

---

## 生成模板

```markdown
# {name} — Production Style

---

## 标题风格

### 句式偏好
{疑问句 / 陈述句 / 感叹句 / 混合，附比例}

### 标题长度
{短（≤10字）/ 中（10-20字）/ 长（>20字），附典型例子}

### 高频套路
{列表，每条附 2-3 个真实标题举例}
- 套路一：{描述}
  - 例："{标题1}"
  - 例："{标题2}"

### 高频词汇
{直接列出他标题里常出现的词}

---

## 开头 Hook（前 15-30 秒）

### 惯用结构
{描述他最常用的开头方式}

### 典型开头举例
> "{开头原文或近似原文}"
> （来自：{视频名/类型}）

### 他不会用的开头方式
{列表，说明为什么}

---

## 内容结构

### 分段方式
{时间轴 / 问题拆解 / 对比 / 故事线 / 混合}

### 典型结构模板
```
{他最常用的内容结构，用伪代码或流程描述}
```

### 金句密度
{大约每隔多少分钟出现一个记忆点/金句}

### 过渡语习惯

## 口播节奏

### 语速
{快 / 中 / 慢，是否有变化节奏}

### 停顿习惯

### 重复强调方式
{他如何强调重点：重复说 / 降速 / 提高音量 / 用"记住这句话"}

---

### 惯用结尾

### 典型结尾话术

### 他不回复的评论类型
```
