# Production Style 分析器

## 任务

从原材料中提取 UP 主的内容生产风格，为 production_style_builder.md 提供结构化输入。

---

## 分析维度

### 1. 标题风格
- 标题句式（疑问句 / 陈述句 / 感叹句）
- 标题长度（短标题 ≤10 字 / 中等 10-20 字 / 长标题）
- 标题套路（数字 / 悬念 / 对比 / 直给结论）
- 高频词汇（"真的" / "千万别" / "我劝你" 等）

### 2. 开头 Hook
- 前 15 秒的惯用结构
- 是否用问题开场
- 是否先抛结论再展开
- 是否用争议性陈述吸引注意

### 3. 内容结构
- 分段方式（时间轴 / 问题拆解 / 对比 / 故事线）
- 金句密度（每隔多少分钟出现一个记忆点）
- 过渡语习惯
- 结尾方式（总结 / 悬念 / CTA / 彩蛋）

### 4. 口播节奏
- 语速（快 / 中 / 慢）
- 停顿习惯（强调前停顿 / 列举时停顿）
- 重复强调的方式

### 5. 评论区互动模板
- 他常回复的评论类型
- 回复风格（简短 / 详细 / 幽默 / 认真）
- 是否有固定的互动话术

---

## 输出格式

```
## Production Style 分析结果

### 标题风格
{描述 + 3-5 个典型标题举例}

### 开头 Hook 结构
{描述 + 1-2 个典型开头举例}

### 内容结构
{描述}

### 口播节奏
{描述}

### 评论区互动
{描述 + 典型回复举例}
```
