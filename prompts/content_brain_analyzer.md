# Content Brain 分析器

## 任务

从原材料中提取 UP 主的内容思维模式，为 content_brain_builder.md 提供结构化输入。

---

## 分析维度

### 1. 选题偏好
- 他反复做的话题类型
- 他明显回避的话题
- 他对热点的态度（追 / 不追 / 反向切）

### 2. 核心观点与立场
- 他在领域内的标志性观点（别人不这么说，他这么说）
- 他的论证习惯（用数据 / 用案例 / 用类比 / 用亲身经历）
- 他的结论风格（给答案 / 留悬念 / 反问观众）

### 3. 知识边界
- 他擅长讲的（有深度、有细节）
- 他讲得浅的（点到为止、不展开）
- 他明确说"不懂"或"不评价"的

### 4. 内容禁区
- 他从不碰的话题（政治 / 竞品 / 私生活 / 其他）
- 他曾经翻车或被骂的内容类型（如有）

---

## 输出格式

```
## Content Brain 分析结果

### 选题偏好
{描述 + 典型选题举例}

### 标志性观点
1. "{观点}" — 论证方式：{方式}
2. ...

### 知识边界
- 擅长：{列表}
- 浅尝：{列表}
- 回避：{列表}

### 内容禁区
{描述}
```
