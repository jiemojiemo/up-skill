# Persona 分析器

## 任务

从原材料中提取 UP 主的人格特征，为 persona_builder.md 提供结构化输入。

---

## 分析维度

### 1. 语气与情绪基调
- 整体语气：严肃 / 轻松 / 毒舌 / 温和 / 激动
- 情绪波动：平稳 / 容易激动 / 冷静克制
- 对观众的态度：平等 / 居高临下 / 像朋友 / 像老师

### 2. 口头禅与高频词
- 直接从字幕/转录中统计出现 3 次以上的特征表达
- 区分：开场白 / 过渡语 / 结尾语 / 情绪词

### 3. 互动风格
- 如何称呼观众（"大家" / "朋友们" / 特定昵称）
- 如何回应质疑（解释 / 反怼 / 忽略 / 幽默化解）
- 如何处理夸奖（谦虚 / 直接接受 / 转移话题）

### 4. 价值观信号
- 反复强调的观点或立场
- 明显的好恶（喜欢什么 / 讨厌什么）
- 对行业/社会的态度

---

## 输出格式

```
## Persona 分析结果

### 语气基调
{描述}

### 口头禅（出现频次排序）
1. "{表达}" — 出现场景：{场景}
2. ...

### 互动风格
{描述 + 原文例子}

### 价值观信号
{描述 + 原文例子}

### 推断标签
{3-5 个行为标签，用于 Layer 0 生成}
```
